## What's a container?

Isolated resources that run on a host machine. They are isolated from each other and from the host machine. They are also isolated from the network.

### What's the kernel?

The kernel is the core of the operating system. It's the part of the operating system that interacts with the hardware. It's the part of the operating system that manages the resources of the machine.

### What's a namespace?

A namespace is a way to isolate resources. It's a way to group resources together. It's a way to group resources together in a way that they can't interact with each other.

### What's a cgroup?

A cgroup is a way to limit the resources that a process can use. It's a way to limit the resources that a process can use in a way that it can't use more than it's allowed to use.

## What's Docker?

Docker is a tool that allows you to run containers. It's a tool that allows you to run containers in a way that they are isolated from each other and from the host machine.

### What's a cluster and nodes?

A cluster is a group of machines that are connected to each other. A cluster is a group of machines that are connected to each other in a way that they can communicate with each other.

A node is a machine that is part of a cluster. A node is a machine that is part of a cluster in a way that it can communicate with the other machines in the cluster.
